# SFMLpp
This repository containt file for both header-only and "regular" version of SFMLpp library. SFMLpp is an addition to SFML library and is used with SFML. SFMLpp alows you to create gui with SFML much more easily. You can create better Textblocks, TextBoxes, Labels, Buttons, Progress Bars, FlowLayoutPanels, Grids and more...
